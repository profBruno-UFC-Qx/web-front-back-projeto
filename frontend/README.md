# Frontend

Aqui devem ficar os arquivos HTML, CSS e Javascript ou Typescript

> [!IMPORTANT]
> Separar os arquivos em pastas é uma boa prática. 

 ``` 
├── css
│   ├── dialog.css
│   └── layout.css
├── js
│   ├── pagina1.js
│   └── pagina2.js
└── paginas
    ├── pagina1.html
    └── pagina2.html
 ```
